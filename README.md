# lane_detection
Lane detection using OpenCV
# Dependencies
OpenCV
## Compiling
```
$ git clone "https://github.com/vidit98/lane_detection/"
$cd lane_detection
$make lane
```
## Run the executables
```
$./lane t22.jpg
```
